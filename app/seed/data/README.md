# seed 内容素材（不入仓库）

本目录的 `*.md` 为私人整理的训练计划/食疗内容素材，**不进 git**（见根 `.gitignore`）。

- `foods_base.csv`（通用食物营养表）随仓库分发，正常入库。
- `plans.py` 在素材缺失时自动跳过对应训练计划，其余 seed（habits / foods / recipes / exercises / settings）不受影响。

如需完整计划内容，把自己的素材按以下文件名放入本目录后重跑 `python -m app.seed`：

| 文件 | 结构要求 |
|---|---|
| `04_训练计划.md` | 头部含 `>` 引用块（安全声明），正文按 `## 计划一` ~ `## 计划五` 分章，文末有 `## 通用注意事项` 章节 |
| `05_无器械综合训练计划.md` | 整文件作为一套计划的 content_md 导入 |
| `06_减脂·早晚徒手计划.md` | 整文件作为一套计划的 content_md 导入 |
